# Linux 简单文件系统

## 功能说明

本程序实现了一个简单的文件系统操作工具，提供以下功能：

1. **列出目录内容** - 列出当前文件夹下所有文件名
2. **打开文件** - 打开已存在的文件或创建新文件
3. **写文件** - 向文件中写入内容
4. **读文件** - 读取并显示文件内容
5. **修改文件权限** - 修改文件权限并提示可修改的权限位
6. **查看文件权限** - 查看当前文件的权限信息
0. **退出** - 退出程序

## 快速启动

```bash
# 1. 编译程序
gcc main.c -o filesystem.exe

# 2. 运行程序
./filesystem.exe
```

## 使用说明

1. 运行程序后，会显示功能菜单
2. 提示用户输入功能号，根据用户输入的功能号执行对应操作
3. 每个功能执行完成后返回主菜单
4. 输入0退出程序

## 权限说明

权限值采用八进制表示：
- **读权限 (Read)**: 4
- **写权限 (Write)**: 2
- **执行权限 (Execute)**: 1

常用权限组合：
- 755: 所有者完全控制，组用户和其他用户读和执行
- 644: 所有者读写，组用户和其他用户只读
- 600: 所有者读写，其他用户无权限

## 注意事项

- 程序使用标准C库函数实现
- Windows环境下需使用MinGW或其他GCC编译器
- Linux环境下直接使用gcc编译即可
- 文件操作会进行基本的错误处理
