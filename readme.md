# 虚拟磁盘文件系统

## 功能说明

本程序实现了一个基于文件的虚拟磁盘文件系统，提供以下功能：

1. **列出文件** - 列出虚拟磁盘中的所有文件
2. **创建文件** - 在虚拟磁盘中创建新文件
3. **写文件** - 向虚拟磁盘中的文件写入内容
4. **读文件** - 读取虚拟磁盘中的文件内容
5. **修改文件权限** - 修改虚拟磁盘中文件的权限
6. **查看文件权限** - 查看虚拟磁盘中文件的权限
7. **格式化虚拟磁盘** - 清空虚拟磁盘中的所有内容
0. **退出** - 退出程序

## 技术实现

- **虚拟磁盘**：使用普通文件作为虚拟磁盘存储介质
- **磁盘大小**：1MB（可通过修改DISK_SIZE宏调整）
- **块大小**：512字节
- **文件表**：存储在虚拟磁盘的起始位置，最多支持16个文件
- **文件结构**：包含文件名、大小、起始块、块数和权限信息

## 快速启动

```bash
# 1. 编译程序
gcc main.c -o filesystem.exe

# 2. 运行程序
./filesystem.exe
```

## 使用说明

1. 运行程序后，会提示输入虚拟磁盘文件名
2. 如果文件不存在，会自动创建新的虚拟磁盘
3. 如果文件存在，会加载已有的虚拟磁盘
4. 显示功能菜单，提示用户输入功能号
5. 根据输入的功能号执行对应操作
6. 每个功能执行完成后返回主菜单
7. 输入0退出程序

## 权限说明

权限值采用八进制表示：
- **读权限 (Read)**: 4
- **写权限 (Write)**: 2
- **执行权限 (Execute)**: 1

常用权限组合：
- 755: 所有者完全控制，组用户和其他用户读和执行
- 644: 所有者读写，组用户和其他用户只读
- 600: 所有者读写，其他用户无权限

## 注意事项

- 程序使用标准C库函数实现，跨平台兼容
- 虚拟磁盘文件存储在当前目录
- 最大支持16个文件，每个文件名最长32个字符
- 文件内容直接存储在虚拟磁盘中，不依赖于操作系统文件系统
- 输入文件内容时，Windows下按Ctrl+Z，Linux下按Ctrl+D结束输入
